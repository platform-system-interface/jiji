# JIJI

JIJI is _Joy In Jotting Images_. 🖼️

It is inspired by (remote) vector graphics protocols from the 60ies and 70ies,
such as [DEC's ReGIS](https://vt100.net/docs/vt3xx-gp/chapter1.html), as well as
the [Plan 9 `/dev/draw`](https://9p.io/magic/man2html/3/draw) device,
[SVG](https://www.w3.org/TR/SVGMobile/REC-SVGTiny12-20081222.pdf), and similar
languages.
